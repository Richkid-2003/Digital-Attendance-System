# Digital Attendance System (HND Electrical Engineering)

## Project Overview
A console-based C++ application to manage student attendance for hour-based lectures.
Developed for the EEE227 Mid-Semester Capstone Project.

## Features
* **Student Management:** Register and view students.
* **Session Management:** Create lecture sessions with dates and times.
* **Attendance Tracking:** Mark students as Present (P), Absent (A), or Late (L).
* **Reports:** View attendance summaries for any session.
* **Data Persistence:** Automatically saves and loads data using text files.

## How to Run
1. Open the project in VS Code.
2. Compile the `main.cpp` file.
3. Run the executable.
4. Use the number keys to navigate the menu.

## File Structure
* `main.cpp`: The source code.
* `students.txt`: Stores registered student data (generated after first run).
* `sessions.txt`: Stores lecture details and attendance records (generated after first run).
